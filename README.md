# shippingbo

Shippingbo — French e-commerce logistics SaaS (OMS / WMS / TMS) with a public REST API for multi-channel order management, warehouse operations, and multi-carrier shipping.

- Website: https://www.shippingbo.com
- Developer Portal: https://developer.shippingbo.com/
- OAuth: https://oauth.shippingbo.com/
- TMS API: https://developer.shippingbo.com/docs/tms-api/4aq6szxp6fsja-get-started-with-shippingbo-tms
